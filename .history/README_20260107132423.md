# Spec-Up-T External Reference Demo 2

This repository is a demo glossary for testing `tref` and `xref` functionality in Spec-Up-T.

## Theme: Plant Lifecycle & Propagation

This glossary focuses on gardening concepts related to plant reproduction and life cycles.

### Terms included:
- **Unique to this glossary**: annual, grafting, propagation, dormancy, germination
- **Shared with other glossaries** (different definitions): soil
- **Same as main boilerplate**: compost, mulch, fertilizer, pruning, seedling, watering

## Usage

This glossary is designed to be referenced from the main Spec-Up-T boilerplate using:

```markdown
[[tref: ExtRef2, dormancy]]
[[xref: ExtRef2, propagation]]
```

Read more about Spec-Up-T in the [documentation](https://trustoverip.github.io/spec-up-t-website/).
